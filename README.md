# TFT Stat Tracker

> Full stack Vue/Express app using the Riot API to track player stats
> [Get your API key from riot](https://developer.riotgames.com/)

## Quick Start

```bash
# Install Express dependencies
npm install

# Install Vue dependencies
cd client
npm install

# Serve
npm run dev

# Build for production
cd client
npm run build
# Gets put in /public
```
